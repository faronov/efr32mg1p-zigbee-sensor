# Documentation Images

## TRÅDFRI Pinout Diagram

Please save the TRÅDFRI ICC-1 pinout diagram as `tradfri-pinout.png` in this directory.

The pinout diagram shows:
- Left side pins: PA0, PA1, PB12, PB13, GND, VCC
- Right side pins: RESET, PF3, PF2 (SWO), PF1 (SWDIO), PF0 (SWCLK), PC11, PC10, PB14, PB15, GND
- Center: SPI Flash connections (CS PB11, MOSI PD15, MISO PD14, CLK PD13)
